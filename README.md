## 📌 About

A responsive student registration web application built with vanilla HTML, CSS, and JavaScript. It allows users to add, edit, and delete student records with data fetched from a REST API.

## ✨ Features

- ✅ Add new students (Name, Email, Phone)
- ✅ Edit existing student records
- ✅ Delete students from the list
- ✅ Fetches real data from JSONPlaceholder API
- ✅ Responsive design
- ✅ Clean dark-themed UI
